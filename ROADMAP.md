# Roadmap

TenantGuard is a config-driven authorization regression testing tool. The items below are planned future work, not current features.

## Planned

- HTML reporter
- OpenAPI-assisted config generator
- GitHub Action wrapper for CI workflows
- pytest plugin for in-process authorization checks
- Optional demo write-method check configuration
- Richer assertion library (additional matchers and helpers)
- API key and cookie authentication support
- Package publishing to PyPI

## Out of scope

- General-purpose vulnerability scanning
- Endpoint discovery or crawling
- Brute force or credential attacks
- Rate-limit bypassing
- Unauthorized testing of third-party systems
